# Theme Specification

> Generated by openlore v1.0.0 on 2026-07-11 03:51

## Purpose

Manages the application's theme settings, including light/dark mode toggling and persistence.

## Requirements

> `web/src/useTheme.ts`

### Requirement: ApplyTheme

> Implementation: `useTheme` in `web/src/useTheme.ts` · confidence: llm

The system SHALL resolves and applies the effective light/dark theme based on user preferences and system settings.

#### Scenario: UserPrefersDarkMode
- **GIVEN** User has set dark mode preference
- **WHEN** applyTheme is called
- **THEN** Dark theme is applied to the root element
- **AND** Theme is stored in localStorage

#### Scenario: SystemPrefersLightMode
- **GIVEN** System prefers light mode and user has not set a preference
- **WHEN** applyTheme is called
- **THEN** Light theme is applied to the root element
- **AND** Theme is stored in localStorage

### Requirement: ToggleTheme

> Implementation: `useTheme` in `web/src/useTheme.ts` · confidence: llm

The system SHALL toggles between light and dark themes and persists the new preference.

#### Scenario: ToggleFromLightToDark
- **GIVEN** Current theme is light
- **WHEN** toggleTheme is called
- **THEN** Dark theme is applied to the root element
- **AND** New theme is stored in localStorage

#### Scenario: ToggleFromDarkToLight
- **GIVEN** Current theme is dark
- **WHEN** toggleTheme is called
- **THEN** Light theme is applied to the root element
- **AND** New theme is stored in localStorage
